# 🌐 Network Topology Analysis and Routing

A Python-based **Computer Networks** project that analyzes network topology, evaluates routing paths, measures latency, and studies bandwidth utilization using graph-based algorithms with **NetworkX**.

---

## 📖 Overview

This project models a computer network as a graph where routers, servers, and devices are represented as nodes and communication links as edges.

The project performs:

- Network topology visualization
- Server and router connectivity analysis
- Shortest path routing
- Latency analysis
- Bandwidth utilization analysis
- Router traffic evaluation

---

## ✨ Features

- 🌐 Network topology visualization
- 🖧 Router and server connectivity analysis
- 📍 Shortest path computation
